# 参数说明
iconClass 此参数为图标类名，可在阿里图标库复制获得每个图标对应的类名，可直接在图标出复制
className 元素类名，可自定义设置字体大小，颜色等样式，覆盖原有样式

# 已注册为全局组件，可直接使用 例:
<icon icon-class="saasicon_zhiwu"></icon>
